# anjumace

Workspace de projetos da Anjumace — designs (Figma), sites, landing pages e web apps.

Cada projeto vive em sua própria pasta na raiz deste workspace.

## Stack padrão

- **React + Vite + Tailwind CSS** para sites e web apps
- Figma para design (telas, componentes, carrosséis, slides)

## Projetos

| Projeto | Tipo | Descrição |
|---------|------|-----------|
| [`design-system`](./design-system) | Design System | Sistema de design completo da Anju Mace (React + Vite + Tailwind): tokens, ~45 componentes, docs e showcase. |

## Como rodar um projeto

```bash
cd <nome-do-projeto>
npm install
npm run dev
```
